# Train Scheduler
This website will help show when your train is arriving. It also shows you how much time remaining until the next train that comes every 15 mins.

## Technologies Used:
1. HTML
2. CSS
3. Javascript
4. Google Fonts
5. Firebase
6. jQuery
7. Bootstrap
